# Project

<br>

### 프로잭트 소개

축구 켜뮤니티 사이트로써 최신 경기의 하이라이트를 볼 수 있을 뿐만 아니라 게시판을 이용하여 사용자들간의 소통이 가능하고, 중고거래 기능을 활용하여 축구 용품이나 여러가지
중고 물품을 거래할 수 있다.

<br>

### 기능
  - 게시판 CRUD
  - 중고거래 CRUD
  - 카카오맵API
  - 회원가입 / 로그인 / 로그아웃
  - 결제 시스템
  - 댓글 / 대댓글 CRUD
